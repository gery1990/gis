服务接口地址
变量 {地图名称} 代表程序要引用的地图名称，地图名称使用地图配置文件配置。
一般地图服务客户端引用URL

URL地址：http://10.173.2.11:80/PGIS_S_TileMapServer/Maps/{地图名称}

地图服务查询接口

接口地址如下： http://10.173.2.11:80/PGIS_S_TileMapServer/Maps/{地图名称}/EzMap

接口参数列表:

参数名称 描述 参数约束
Service 服务类型 必须的，且必须是“getImage”
Type 图像像素格式 必须的，且必须是“RGB”
ZoomOffset 缩放等级偏移量 非必须的，默认值是“0”
Col 图片列号 必须的
Row 图片行号 必须的
Zoom 图片缩放等级 必须的
V 版本 必须是“0.3”

典型的请求格式如下：

http://10.173.2.11:80/PGIS_S_TileMapServer/Maps/{地图名称}/EzMap?Service=getImage&Type=RGB&ZoomOffset=3&Col=232&Row=79&Zoom=5&V=0.3
地图快照服务查询接口

接口地址如下：http://10.173.2.11:80/PGIS_S_TileMapServer/Maps/{地图名称}/EzMap

接口参数列表:

参数名称 描述 参数约束
Service 服务类型 必须的，且必须是“getRectImg”
minx 地图X轴最小区域边界 必须的
miny 地图Y轴最小区域边界 必须的
maxx 地图X轴最大区域边界 必须的
maxy 地图Y轴最大区域边界 必须的
zoom 图片缩放等级 必须的
result 输出文件格式 必须的,可选择（zip|html|image）

典型的请求格式如下：

http://10.173.2.11:80/PGIS_S_TileMapServer/Maps/{地图名称}/EzMap?Service=getRectImg&result=html&minx=116.38393&miny=39.91252&maxx=116.40194&maxy=39.93126&zoom=3
