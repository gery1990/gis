地图工具概述

东方网力地图主要提供地图测量工具和几何绘制工具：

- 测量工具：通过该工具可以实现距离（两点之间）测量和面积（多个点闭合而成）测量；

- 几何绘制工具：通过该工具可以实现线、矩形、圆形、多边形的绘制；

测量工具

						//测量工具
						var tool = new NPMapLib.Tools.MeasureTool(map, {
								lengthUnit: NPMapLib.MAP_UNITS_METERS, //长度单位
								areaUnit: NPMapLib.MAP_UNITS_SQUARE_KILOMETERS,	//面积单位
								mode: NPMapLib.MEASURE_MODE_DISTANCE //测量模式
							});
		
						//操作
						/*测距离*/
						tool.setMode(NPMapLib.MEASURE_MODE_DISTANCE);
						/*测面积*/
						tool.setMode(NPMapLib.MEASURE_MODE_AREA);
						/*开启*/
						tool.startUp();
						/*取消*/
						tool.cancel();

绘制工具

						//绘制工具
						var tool = new NPMapLib.Tools.DrawingTool(map);

						//操作
						/*画线*/
						tool.setMode(NPMapLib.DRAW_MODE_POLYLINE, callBackMethod);
						/*画矩形*/
						tool.setMode(NPMapLib.DRAW_MODE_RECT, callBackMethod);
						/*画圆形*/
						tool.setMode(NPMapLib.DRAW_MODE_CIRCLE, callBackMethod);
						/*画多边形*/
						tool.setMode(NPMapLib.DRAW_MODE_POLYLGON, callBackMethod);
						/*开启*/
						tool.startUp(callBackMethod);
						/*取消*/
						tool.cancel();
