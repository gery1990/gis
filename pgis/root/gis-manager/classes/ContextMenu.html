构造函数
构造函数 描述
ContextMenu() 创建右键菜单实例。
方法
方法 返回值 描述
addItem(item:ContextMenuItem) none 添加菜单项。
getItem(index:Number) ContextMenuItem 返回指定索引位置的菜单项。
第一个添加菜单项的index为0，如此递增。
removeItem(index:Number) none 移除菜单项。
addSeparator() none 添加分隔符。
removeSeparator(index:Number) none 移除分隔符。
show() none 显示。
hide() none 隐藏。
