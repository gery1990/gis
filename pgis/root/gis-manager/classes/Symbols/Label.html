构造函数
构造函数 描述
Label(content:String, opts:Array) 创建文本标注实例。
content：内容。
opts={
      offset:new NPMapLib.Geometry.Size(0, 0), //文本标注的位置偏移值
      position: new NPMapLib.Geometry.Point(0,0) //文本标注的地理位置
};
方法
方法 返回值 描述
setStyle(styles:Array) none 设置文本标注样式，该样式将作用于文本标注的容器元素上。
{
      fontSize:12, //文字大小
      fontFamily:'宋体', //字体
      color:'red', //文字前景色
      bgColor:'#FFFFFF', //背景色
      bordColor:'#CCCCFF', //边框颜色
      borderSize:1, //边框宽度
      align:'center', //对方方式
      isBold:false //是否粗体
};
getStyle() Array 获取文本标注样式。
setContent(content:String) none 设置文本标注的内容。
getContent() String 获取内容。
setPosition(position:Point) none 设置文本标注坐标。
getPosition() Point 获取标注位置。
setOffset(offset:Size) none 设置文本标注的偏移值。
getOffset() 返回文本标注的偏移值。
setTitle(title:String) none 设置文本标注的标题。
getTitle() String 返回文本标注的标题。
addEventListener(event:enum, handler:Function) none 添加事件监听函数。
removeEventListener(event:enum) none 移除事件监听函数。
事件
事件类型 回调函数 描述
NPMapLib.LABEL_EVENT_CLICK target(point:Point) 点击标注后会触发此事件。
NPMapLib.LABEL_EVENT_DBLCLICK target(point:Point) 双击标注后会触发此事件。
NPMapLib.LABEL_EVENT_RIGHT_CLICK target(point:Point) 右键点击标注时触发此事件。
NPMapLib.LABEL_EVENT_MOUSE_DOWN target(point:Point) 鼠标在标注上按下触发此事件。
NPMapLib.LABEL_EVENT_MOUSE_UP target(point:Point) 鼠标在标注上释放触发此事件。
NPMapLib.LABEL_EVENT_MOUSE_OVER target(point:Point) 当鼠标进入标注区域时会触发此事件。
NPMapLib.LABEL_EVENT_MOUSE_OUT target(point:Point) 鼠标离开标注时触发此事件。
NPMapLib.LABEL_EVENT_DRAG_START target() 开始拖拽标注时触发此事件。
NPMapLib.LABEL_EVENT_DRAG_END target() 拖拽结束时触发此事件。
