命名空间：NPMapLib.Symbols    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
Animation(mapId:Number, overLay:Overlay, opts:Array 创建一个图像标注实例，继承自Overlay。
mapId：地图id。
overLay：覆盖物。
opts={
      speed: 100, //速度
      isReturn: true, //是否往返：若为true，则来回推演
      isRepeat: true //是否重复：若为true，则反复推演
};
方法
方法 返回值 描述
setPath(path:Polyline) none 设置推演行径路线。
getPath() Icon 返回推演行径路线。
getOverlay() Overlay 返回覆盖物对象。
setRepeat(isRepeat:Boolean) none 设置是否重复。
setReturn(isReturn:Boolean) none 设置是否返回。
setSpeed(speed:Number) none 设置速度。
start() none 启动推演。
pause() none 暂停推演。
stop() none 停止推演。
addEventListener(event:enum,handler:Function) none 添加事件监听函数。
removeEventListener(event:enum) none 移除事件监听函数。
事件
事件类型 回调函数 描述
NPMapLib.ANIMATION_EVENT_START target(point:Point) 启动推演时会触发此事件。
NPMapLib.ANIMATION_EVENT_PAUSE target(point:Point) 暂停推演时会触发此事件。
NPMapLib.ANIMATION_EVENT_STOP target(point:Point) 停止推演时会触发此事件。
NPMapLib.ANIMATION_EVENT_MOVING target(point:Point) 覆盖物准备移动时触发此事件。
NPMapLib.ANIMATION_EVENT_MOVED target(point:Point) 覆盖物移动后触发此事件。
