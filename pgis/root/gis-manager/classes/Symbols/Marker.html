命名空间：NPMapLib.Symbols    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
Marker(point:Point, opts:Array 创建一个图像标注实例，继承自Overlay。
point：指定了图像标注所在的地理位置。
opts={
      offset: new NPMapLib.Geometry.Size(0,0), //标注的位置偏移值
      icon:null, //标注所用的图标对象
      enableDragging:true //是否启用拖拽，默认为false
};
方法
方法 返回值 描述
setIcon(icon:Icon) none 设置标注所用的图标对象。
getIcon() Icon 返回标注所用的图标对象。
setPosition(point:Point) none 设置标注的地理坐标。
getPosition() Point 返回标注的地理坐标。
setOffset(offset:Size) none 设置标注的偏移值。
getOffset() Size 返回标注的偏移值。
setLabel(label:Label) none 为标注添加文本标注。
getLabel() Label 返回标注的文本标注。
showLabel() none 显示文本。
hideLabel() none 隐藏文本。
setTitle(title:Title) none 设置标注的标题，当鼠标移至标注上时显示此标题。
getTitle() Title 返回标注的标题。
enableDragging() none 开启标注拖拽功能。
disableDragging() none 关闭标注拖拽功能。
addContextMenu(menu:ContextMenu) none 添加右键菜单。
removeContextMenu(menu:ContextMenu) none 移除右键菜单。
addEventListener(event:enum,handler:Function) none 添加事件监听函数。
removeEventListener(event:enum) none 移除事件监听函数。
事件
事件类型 回调函数 描述
NPMapLib.MARKER_EVENT_CLICK target(point:Point) 点击标注后会触发此事件。
NPMapLib.MARKER_EVENT_DBLCLICK target(point:Point) 双击标注后会触发此事件。
NPMapLib.MARKER_EVENT_RIGHT_CLICK target(point:Point) 右键点击标注时触发此事件。
NPMapLib.MARKER_EVENT_MOUSE_DOWN target(point:Point) 鼠标在标注上按下触发此事件。
NPMapLib.MARKER_EVENT_MOUSE_UP target(point:Point) 鼠标在标注上释放触发此事件。
NPMapLib.MARKER_EVENT_MOUSE_OVER target(point:Point) 当鼠标进入标注区域时会触发此事件。
NPMapLib.MARKER_EVENT_MOUSE_OUT target(point:Point) 鼠标离开标注时触发此事件。
NPMapLib.MARKER_EVENT_DRAG_START target() 开始拖拽标注时触发此事件。
NPMapLib.MARKER_EVENT_DRAG_END target() 拖拽结束时触发此事件。
