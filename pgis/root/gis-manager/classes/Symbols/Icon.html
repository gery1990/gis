命名空间：NPMapLib.Symbols    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
Icon(imageUrl:String, size:Size, opts:Array) 创建一个图标对象。
imageUrl:图标路径。
size:图标大小。
opts={
      anchor: new NPMapLib.Geometry.Size(0, 0) //图标的定位锚点
};
方法
方法 返回值 描述
setImageUrl(imageUrl：String) none 设置图片资源的路径。
getImageUrl() String 返回图片资源的路径。
setImageSize(size:Size) none 设置图标可视区域的大小。
getImageSize() Size 返回图标可视区域的大小。
setAnchor(anchor：Size) none 设置图标定位点相对于其左上角的偏移值。
getAnchor() Size 返回图标定位点相对于其左上角的偏移值。
