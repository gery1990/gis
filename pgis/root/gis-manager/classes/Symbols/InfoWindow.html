命名空间：NPMapLib.Symbols    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
InfoWindow(title:String, content：String|HTMLElement, opts:Array 创建一个信息窗实例。
title：标题。
content：内容。支持HTML内容。默认信息窗不显示。
opts={
      width: 260, //信息窗宽度，单位像素
      height: 180, //信息窗高度，单位像素
      offset: new NPMapLib.Geometry.Size(0,0), //信息窗位置偏移值
      arrow: true, //是否带箭头
      arrowStyle: NPMapLib.LINE_ARROW_TYPE_NULL, //线的箭头样式
      enableCloseOnClick: false, //是否开启点击地图关闭信息窗口（默认开启）
};
方法
方法 返回值 描述
open(x:Number, y:Number) none 打开。
close() none 关闭。
setWidth(width:Number) none 设置信息窗口的宽度，单位像素。
setHeight(height:Number) none 设置信息窗口的高度，单位像素。
setTitle(title:String) none 设置信息窗口标题。
getTitle() String 返回信息窗口标题。
setContent(content:String) none 设置信息窗口内容。
getContent() String 返回信息窗口内容。
getPosition() Point 返回信息窗口的位置。
isOpen Boolean 返回信息窗口的打开状态。
enableCloseOnClick() none 开启点击地图时关闭信息窗口。
disableCloseOnClick none 关闭点击地图时关闭信息窗口。
addEventListener(event:enum,handler:Function none 添加事件监听函数。
removeEventListener(event:enum) none 移除事件监听函数。
