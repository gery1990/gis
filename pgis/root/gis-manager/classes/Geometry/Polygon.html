命名空间：NPMapLib.Geometry    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
Polygon(points:Array, opts:Array) 创建多边形实例，继承自Overlay。
opts={
      color: "blue", //边线颜色
      fillColor: "red", //填充颜色
      weight: 2, //边线的宽度，以像素为单位
      opacity: 1.0, //边线透明度，取值范围0-1
      fillOpacity: 0.5, //填充的透明度，取值范围0-1
      lineStyle: NPMapLib.LINE_TYPE_SOLID, //边线的样式
      enableEditing: false, //是否启用编辑，默认为false
      enableClicking: true, //是否响应点击事件，默认为true
};
方法
方法 返回值 描述
setPath(path:Array) none 设置多段线的点数组。
getPath() Array 返回多段线的点数组。
setColor(color:String) none 设置多段线的边线颜色。
getColor() String 返回多段线的边线颜色。
setFillColor(color:String) none 设置多段线的填充颜色。
getFillColor() String 返回多段线的填充颜色。
setOpacity(opacity:Number) none 设置透明度，取值范围0 - 1。
getOpacity() Number 返回透明度。
setFillOpacity(opacity:Number) none 设置多段线的填充透明度，取值范围0 - 1。
getFillOpacity() Number 返回多段线的填充透明度。
setWeight(weight:Number) none 设置多段线的线宽，范围为大于等于1的整数。
getWeight() Number 返回多段线边线的宽度。
setLineStyle(lineStyle:enum) none 设置当前线样式。
实线（默认）：NPMapLib.LINE_TYPE_SOLID
虚线：NPMapLib.LINE_TYPE_DASH
点：NPMapLib.LINE_TYPE_DOT
虚线（中间夹杂点点)：NPMapLib.LINE_TYPE_DASHDOTDOT
虚线（中间夹杂点)：NPMapLib.LINE_TYPE_DASHDOT
短虚线：NPMapLib.LINE_TYPE_SHORTDASH
短点线：NPMapLib.LINE_TYPE_SHORTDOT
短虚线（中间夹杂点点）：NPMapLib.LINE_TYPE_SHORTDASHDOTDOT
短虚线（中间夹杂点）：NPMapLib.LINE_TYPE_SHORTDASHDOT
长虚线：NPMapLib.LINE_TYPE_LONGDASH
长虚线（中间夹杂点）：NPMapLib.LINE_TYPE_LONGDASHDOT
getLineStyle() enum 返回当前线样式。
getExtent() Extent 返回覆盖物的地理区域范围。
enableEditing() none 开启编辑功能。
disableEditing() none 关闭编辑功能。
setPositionAt(index:Number, point:Point) none 修改指定位置的坐标。
index-指定点的坐标编号；point:修改后的坐标值。
addEventListener(event:enum, handler:Function) 添加事件监听函数。
removeEventListener(event:enum) none 移除事件监听函数。
事件
事件类型 回调函数 描述
NPMapLib.POLYGON_EVENT_CLICK target(point:Point) 点击标注后会触发此事件。
NPMapLib.POLYGON_EVENT_DBLCLICK target(point:Point) 双击标注后会触发此事件。
NPMapLib.POLYGON_EVENT_RIGHT_CLICK target(point:Point) 右键点击标注时触发此事件。
NPMapLib.POLYGON_EVENT_MOUSE_DOWN target(point:Point) 鼠标在标注上按下触发此事件。
NPMapLib.POLYGON_EVENT_MOUSE_UP target(point:Point) 鼠标在标注上释放触发此事件。
NPMapLib.POLYGON_EVENT_MOUSE_OVER target(point:Point) 当鼠标进入标注区域时会触发此事件。
NPMapLib.POLYGON_EVENT_MOUSE_OUT target(point:Point) 鼠标离开标注时触发此事件。
NPMapLib.POLYGON_EVENT_LINE_UPDATE target() 覆盖物的属性发生变化时触发。
