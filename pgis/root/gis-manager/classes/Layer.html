说明：图层的抽象基类，所有图层均继承基类的方法。此类不可实例化。
命名空间：NPMapLib    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
属性
属性 类型 描述
id Number 唯一标识。
mapId Number 地图id。
方法
方法 返回值 描述
getUrl() String 获取图层路径。
setUrl(url:String) none 设置图片路径。
isVisible() Boolean 判断是否显示。
setName(name:String) none 设置图层名称。
getName() String 获取图层名称。
setMinLevel(level:Number) none 设置最小显示级别。
getMinLevel() Number 获取最小显示级别。
setMaxLevel(level:Number) none 设置最大显示级别。
getMaxLevel() Number 获取最大显示级别。
show() none 显示图层。
hide() none 隐藏图层。
