说明：控件的抽象基类，所有控件均继承基类的方法。此类不可实例化。
命名空间：NPMapLib    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
属性
属性 类型 描述
id Number 唯一标识。
controlType Enum 控件类型。
未知：NPMapLib.CONTROL_TYPE_UNKNOWN;
版权控件：NPMapLib.CONTROL_TYPE_COPYRIGHT;
导航控件：NPMapLib.CONTROL_TYPE_NAVIGATION;
鹰眼控件：NPMapLib.CONTROL_TYPE_OVERVIEW;
比例尺控件：NPMapLib.CONTROL_TYPE_SCALE;
mapId Number 所属地图id。
方法
方法 返回值 描述
isVisible() Boolean 判断是否显示。
setOffset(offset:Size) none 设置控件停靠的偏移量。
getOffset() Size 返回控件停靠的偏移量。
setAnchor(anchor:enum) none 设置控件停靠位置。
左上角：NPMapLib.CONTROL_ANCHOR_TOP_LEFT;
右上角：NPMapLib.CONTROL_ANCHOR_TOP_RIGHT;
左下角：NPMapLib.CONTROL_ANCHOR_BOTTOM_LEFT;
右下角：NPMapLib.CONTROL_ANCHOR_BOTTOM_RIGHT;
getAnchor() enum 获取控件停靠位置。
show() none 显示。
hide() none 隐藏。
none 设置控件停靠的偏移量。
setAnchor(anchor:enum) none 设置停靠位置。
