说明：覆盖物的抽象基类。此类不可实例化。
命名空间：NPMapLib    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
Overlay() 创建叠加实例对象。
属性
属性 类型 描述
id Number 唯一标识。
overlayType enum 覆盖物类型。
未知：NPMapLib.OVERLAY_TYPE_UNKNOWN；
多段线：NPMapLib.OVERLAY_TYPE_POLYLINE；
多边形：NPMapLib.OVERLAY_TYPE_POLYGON；
圆形：NPMapLib.OVERLAY_TYPE_CIRCLE；
文本：NPMapLib.OVERLAY_TYPE_LABEL；
图像标注：NPMapLib.OVERLAY_TYPE_MARKER；
mapId Number 所属地图id。
方法
方法 返回值 描述
isVisible() boolean 是否显示。
show() none 显示。
hide() none 隐藏。
flash() none 闪烁3次
flash2(times:Number) none 闪烁。times：闪烁次数
getZIndex() Number 获取当前对象在图层中的叠加次序。
setZIndex(zIndex:Number) none 设置当前对象在图层中的叠加次序。
rotate(angle:Number) none 旋转。
angle为弧度。
