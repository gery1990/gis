方法 返回值 描述

getChildItem(index:Number) none 返回指定索引位置的菜单项。
第一个添加子菜单项的index为0，如此递增。
addSeparator() none 添加分隔符。
removeSeparator(index:Number) none 移除分隔符。
enable() none 启用。
disable() none 禁用。
