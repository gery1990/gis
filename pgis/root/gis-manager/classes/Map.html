命名空间：NPMapLib    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
Map(container:String|HTMLElement, opts:Array) 在指定的容器内创建地图实例。
opts={
        minZoom: 0, //地图允许展示的最小级别
        maxZoom: 10 //地图允许展示的最大级别
};
属性
属性 类型 描述
id Number 唯一标识。
地图配置
方法 返回值 描述
enableScrollWheelZoom() none 启用鼠标滚轮放大缩小，默认启用。
disableScrollWheelZoom() none 禁用鼠标滚轮放大缩小。
enableDoubleClickZoom() none 启用双击放大，默认启用。
disableDoubleClickZoom() none 禁用双击放大。
enableKeyboard() none 启用键盘操作，默认启用。键盘的上、下、左、右键可连续移动地图。同时按下其中两个键可使地图进行对角移动。PgUp、PgDn、Home和End键会使地图平移其1/2的大小。+、-键会使地图放大或缩小一级。
disableKeyboard() none 禁用键盘操作。
enableInertialDragging() none 启用地图惯性拖拽，默认禁用。
disableInertialDragging() none 禁用地图惯性拖拽。
setCursor(cursor:String) none 设置地图鼠标指针样式。
getCursor() String 返回地图鼠标指针样式。
setMinZoom(zoom:Nnmber) none 设置地图允许的最小级别。取值不得小于地图类型所允许的最小级别。
setMaxZoom(zoom:Number) none 设置地图允许的最大级别。取值不得大于地图类型所允许的最大级别。
地图状态方法
方法 返回值 描述
getExtent() Extent 返回地图可视区域，以地理坐标表示。
getCenter() Point 返回地图当前中心点。
getDistance(start:Point, end:Point) Number 返回两点之间的距离，单位是米。
getSize() Size 返回地图视图的大小，以像素表示。
getZoom() Number 返回地图当前缩放级别。
修改地图状态
方法 返回值 描述
centerAndZoom(center:Point,zoom:Number) none 地图根据指定的点和级别进行对中。center只能在地图的fullExtent范围内，zoom只能在地图的最大显示级别和最小显示级别之间。
pan() none 地图平移。
panTo(center:Point) none 将地图的中心点更改为给定的点。
panByPixel(x:Number,y:Number) none 将地图在水平位置上移动x像素，垂直位置上移动y像素。
reset() none 重新设置地图，恢复地图初始化时的中心点和级别
setCenter(center:Point) none 设置地图中心点。
switchLayer(index:Number) none 切换图层。
index为添加到地图时的顺序。
setZoom(zoom:Number) none 将地图缩放到指定的缩放等级，中心点坐标不变。
fullExtent() none 全图。
zoomIn() none 拉框放大。
zoomOut() none 拉框缩小。
zoomInFixed() none 固定放大一个级别。
zoomOutFixed() none 固定缩小一个级别。
zoomToExtent(extent:Extent) none 地图缩放到指定范围
控件
方法 返回值 描述
addControl(control:Control) none 将控件添加到地图，一个控件实例只能向地图中添加一次。
removeControl(control:Control) none 从地图中移除控件。如果控件从未被添加到地图中，则该移除不起任何作用。
getContainer() HTMLElement 返回地图的容器元素
覆盖物
方法 返回值 描述
addOverlay(overlay:Overlay) none 将覆盖物添加到地图中，一个覆盖物实例只能向地图中添加一次。
addOverlays(overlays:Array) none 将多个覆盖物添加到地图中。
removeOverlay(overlay:Overlay) none 从地图中移除覆盖物。如果覆盖物从未被添加到地图中，则该移除不起任何作用。
removeOverlays(overlays:Array) none 从地图中移除覆盖物。如果覆盖物从未被添加到地图中，则该移除不起任何作用。
getOverlays() Array 返回地图上的所有覆盖物。
clearOverlays() none 清除地图上所有覆盖物。
地图图层
方法 返回值 描述
addLayer(layer:Layer) none 添加一个图层。
addLayers(layers:Array) none 添加图层组。
removeLayer(id:Number) none 移除一个图层，根据图层id。
getLayer(id:Number) Layer 获取图层，根据图层id。
坐标变换
方法 返回值 描述
pixelToPoint(pix:Pixel) Point 像素坐标转换为经纬度坐标。
pointToPixel(point:Point) Pixel 经纬度坐标转换为像素坐标。
其他
方法 返回值 描述
addEventListener(event:enum, handle:Function) none 添加事件。
event:事件类型。
handle:回调函数。
removeEventListener(event:enum) none 移除事件
事件
事件类型 回调函数 描述
NPMapLib.MAP_EVENT_CLICK target(point:Point) 左键单击地图时触发此事件。
NPMapLib.MAP_EVENT_DBLCLICK target(point:Point) 双击地图时会触发此事件。
NPMapLib.MAP_EVENT_RIGHT_CLICK target(point:Point) 右键单击地图时触发此事件。
NPMapLib.MAP_EVENT_MOUSE_MOVE target(point:Point) 鼠标在地图区域移动过程中触发此事件。
NPMapLib.MAP_EVENT_MOUSE_OVER target(point:Point) 鼠标移入地图区域时触发此事件。
NPMapLib.MAP_EVENT_MOUSE_OUT target(point:Point) 鼠标移出地图区域时触发此事件。
NPMapLib.MAP_EVENT_ZOOM_START target(zoomLevel:Number) 地图更改缩放级别开始时触发触发此事件。
NPMapLib.MAP_EVENT_ZOOM_END target(zoomLevel:Number) 地图更改缩放级别结束时触发触发此事件。
NPMapLib.MAP_EVENT_ADD_OVERLAY target() 当使用Map.addOverlay()方法向地图中添加单个覆盖物时会触发此事件。
NPMapLib.MAP_EVENT_REMOVE_OVERLAY target() 当使用Map.removeOverlay()方法移除单个覆盖物时会触发此事件。
NPMapLib.MAP_EVENT_CLEAR_OVERLAYS target() 当使用Map.clearOverlays()方法一次性移除全部覆盖物时会触发此事件。
NPMapLib.MAP_EVENT_ADD_CONTROL target() 当使用Map.addControl()方法向地图中添加单个控件时会触发此事件。
NPMapLib.MAP_EVENT_REMOVE_CONTROL target() 当使用Map.removeControl()方法移除单个控件时会触发此事件。
NPMapLib.MAP_EVENT_DRAG_START target() 开始拖拽地图时触发。
NPMapLib.MAP_EVENT_DRAG_END target() 停止拖拽地图时触发。
NPMapLib.MAP_EVENT_RESIZE target(size:Size) 地图可视区域大小发生变化时会触发此事件。
