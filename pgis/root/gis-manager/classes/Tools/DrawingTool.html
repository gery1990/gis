命名空间：NPMapLib.Tools    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
DrawingTool(map:Map) 创建几何图形绘制工具实例。
map：地图对象。
属性
属性 类型 描述
mapId Number 地图容器实例Id。
方法
方法 返回值 描述
setMode(mode:enum, callback) none 设置绘制模式。
不绘制：NPMapLib.DRAW_MODE_NONE;
画矩形：NPMapLib.DRAW_MODE_RECT;
画圆：NPMapLib.DRAW_MODE_CIRCLE;
画多段线：NPMapLib.DRAW_MODE_POLYLINE;
画面：NPMapLib.DRAW_MODE_POLYLGON;
getMode() enmu 获取绘制模式。
startUp() none 启动绘制。
cancel() none 取消绘制。
