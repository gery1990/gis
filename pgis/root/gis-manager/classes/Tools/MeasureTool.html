命名空间：NPMapLib.Tools    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
MeasureTool(map:Map, opts:Array) 创建测量工具实例。
map：地图对象。
opts={
      lengthUnit: NPMapLib.MAP_UNITS_METERS, //长度单位，默认是米
      areaUnit: NPMapLib.MAP_UNITS_SQUARE_KILOMETERS, //面积单位，默认为平方千米
      mode: NPMapLib.MEASURE_MODE_DISTANCE //测量模式，默认为测量长度
};
属性
属性 类型 描述
mapId Number 地图容器实例Id。
方法
方法 返回值 描述
getLengthUnit() enum 获取长度单位。
setLengthUnit(unit:enum) none 设置长度单位。
未知：NPMapLib.MAP_UNITS_UNKNOWN;
厘米：NPMapLib.MAP_UNITS_CENTIMETERS;
经纬度：NPMapLib.MAP_UNITS_DECIMAL_DEGREES;
度分秒：NPMapLib.MAP_UNITS_DEGREE_MINUTE_SECONDS;
分米：NPMapLib.MAP_UNITS_DECIMETERS;
英尺：NPMapLib.MAP_UNITS_FEET;
英寸：NPMapLib.MAP_UNITS_INCHES;
公里：NPMapLib.MAP_UNITS_KILOMETERS;
米：NPMapLib.MAP_UNITS_METERS;
英里：NPMapLib.MAP_UNITS_MILES;
毫米：NPMapLib.MAP_UNITS_MILLIMETERS;
海里：NPMapLib.MAP_UNITS_NAUTICAL_MILES;
点：NPMapLib.MAP_UNITS_POINTS;
码：NPMapLib.MAP_UNITS_YARDS;
英亩：NPMapLib.MAP_UNITS_ACRES;
getAreaUnit() enum 获取面积单位。
setAreaUnit(unit:enum) none 设置面积单位。面积单位参数如下：
公亩：NPMapLib.MAP_UNITS_ARES;
平方公里：NPMapLib.MAP_UNITS_SQUARE_KILOMETERS;
平方英里：NPMapLib.MAP_UNITS_SQUARE_MILES;
平方英尺：NPMapLib.MAP_UNITS_SQUARE_FEET;
平方米：NPMapLib.MAP_UNITS_SQUARE_METERS;
公顷：NPMapLib.MAP_UNITS_HECTARES;
平方码：NPMapLib.MAP_UNITS_SQUARE_YARDS;
平方英寸：NPMapLib.MAP_UNITS_SQUARE_INCHES;
平方毫米：NPMapLib.MAP_UNITS_SQUARE_MILLIMETERS;
平方厘米：NPMapLib.MAP_UNITS_SQUARE_CENTIMETERS;
平方分米：NPMapLib.MAP_UNITS_SQUARE_DECIMETERS;
setMode(mode:enum) mode 设置测量模式。测量模式参数如下：
测量距离：NPMapLib.MEASURE_MODE_DISTANCE
测量面积：NPMapLib.MEASURE_MODE_AREA
测量区域：NPMapLib.MEASURE_MODE_LOCATION
getMode() enum 获取测量模式。
startUp() none 启动。
cancel() none 取消。
