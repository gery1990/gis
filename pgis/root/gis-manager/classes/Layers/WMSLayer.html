构造函数
构造函数 描述
WMSLayer(url:String, name:String, opts:Array) 创建WMS缓存地图图层实例，继承自Layer。
url：图层服务地址；
name：图层名称；
opts={
        layerName: "", //图层标识
        projection: "EPSG:4326", //投影方式
        fullExtent: [0, 0, 0, 0], //全图坐标范围
        minLevel: 0, //最小显示级别
        maxLevel: 10, //最大显示级别
        tiled: false, //是否缓存
        format: 'image/png', //缓存图片格式
        mapUnitPixels: 256, //缓存图片像素
        buffer: 0, //图片加载方式，0--快速下载，2--双螺旋下载[默认为0]
        reproject:true, //是否进行重投影，默认为true
        isBaseLayer: true, //是否作为基础图层，备注：基础图层无法控制显示隐藏
}
