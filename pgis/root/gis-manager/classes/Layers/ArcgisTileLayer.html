构造函数
构造函数 描述
ArcgisTileLayer(url:String, name:String, opts:Array) 创建ArcGIS在线缓存地图图层实例，继承自Layer。
url：图层服务地址；
name：图层名称；
opts={
        centerPoint: [0, 0], //中心点
        fullExtent: [0, 0, 0, 0], //全图坐标范围
        minLevel: 0, //最小显示级别
        maxLevel: 10, //最大显示级别
        zoomOffset: 10, //缩放级别偏移量
        tilePixels: 256, //切片规格
        zoomLevelSequence: 2, //设置地图比例尺级别是降序还是升序
        initResolution: [a,b,c...], //分辨率
        origin: [0, 0] //坐标原点
}
