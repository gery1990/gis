命名空间：NPMapLib.Layers    (PS：力图精简，以下描述皆省略命名空间，详见相关类的说明页面)
构造函数
构造函数 描述
EzMapOffLineLayer(url:String, name:String, opts:Array) 创建PGis离线缓存地图图层实例，继承自Layer。
url：图层服务地址；
name：图层名称；
opts={
        centerPoint：[0, 0],//图层的中心坐标
        fullExtent：[0, 0, 0, 0],//全图范围
        minLevel：0,//图层显示的最小级别
        maxLevel：10，//图层显示的最大级别
        zoomOffset：0，//设置地图级别的偏移量
        zoomLevelSequence：2//设置地图比例尺级别是降序还是升序的
}
方法
方法 返回值 描述
setTileImageFormat(imageFormat:enum) none 设置切片图片格式。
.png格式：NPMapLib.IMAGE_FORMAT_PNG
.jpg格式：NPMapLib.IMAGE_FORMAT_JPEG
.bmp格式：NPMapLib.IMAGE_FORMAT_BMP
